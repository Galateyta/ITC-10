3.Task: Grub-ի կոնֆիգուրացիոն նիշքի մեջ կատարեք փոփոխություններ այնպես, որ grub-ի պատուհանը երևա 10 վայրկյան, անջատեք օպերացիոն համակարգի ներբեռնման ժամանակ երևացող նկարը, նշիչը ավտոմատ պետք է սեղմվի 20 վայրկյան հետո։ Նկարագրեք ձեր կատարած քայլերը։  Done

Կատարոլ եմ հետևյալ քայլերը՝

    1.Բացել եմ Grub֊ի կոնֆիգուրացիոն ֆայլը՝
        sudo vim /etc/default/grub
    2.Ֆայլում կատարել եմ հետևալ վերագրումները՝
        GRUB_HIDDEN_TIMEOUT=10
        GRUB_CMDLINE_LINUX_DEFAULT=""
        GRUB_TIMEOUT=20
    3.sudo update-grub

7.Task:  Ինչ է PPA֊ն։ Փորձեք ավելացնել java պարունակող PPA֊ն, որից հետո տեղադրել java-ն: Done

PPA -ն ստանդարտ ռեպոզիտորիա է պահում է ծրագրերի ստաբիլ վերսիաները, որոնք դուրս եկած են եղել մեր Ubuntu-ի վերսիաի դուրս գալու ժամանակ: Հետո այնտեղ ավելացնում են թարմեցված ապահովությունները: Կողմնակի ծրագրավորողներին դժվար է ստանդարտ ռեպոզիտորիաների մեջ ավելացնել իրենց հավելվածները: Իրենց օգնության է գալիս Canonical - ստեղծող -Ubuntu ստեղծել է ռեպոզիտորիա https://launchpad.net/, որը ընդունված է անվանել PPA(Personal Packages Archive -Արխիվ - անձնային արխիվի փաթեթներ)

	sudo add-apt-repository ppa:webupd8team/java
	sudo apt-get install oracle-java7-installer
